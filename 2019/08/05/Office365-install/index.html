Office365 安装

关于Office 365 安装步骤

手工修改配置文件方法

  1. 将office365 离线安装包解压并拷贝至磁盘剩余较多的目录（不一定要是C盘，现在的Office365 安装；路径默认C盘，并且不可选择）。
  2. 选择对应操作系统x86和x64系统对应安装包的目录，找到对应的配置文件 configuration-Office365-x64.xml（如 64位系统的安装目录）。
  3. 用记事本或者可以打开该文件的编辑器打开，找到对应的XML 节点属性值 SourcePath，对应该值和该配置文件所在的目录是否一致（复制当前目录路径方法：选中文件夹导航路径，全选复制）。如果值不一致，则修改成对应的路径，一致则不需要修改。为防止安装的时候报错，在SourcePath同一节点的属性内加一个值
